About the Project

Understanding what this project does and why it matters

🎯 Project Overview

This project is a Text-to-Speech (TTS) application built using Python. It takes text input from the user and converts it into natural-sounding human speech using Microsoft Edge's Neural AI voices.


The program supports multiple languages and voices including Hindi (Male & Female) and English (Male & Female with US & UK accents). Users can type any text — in Hindi or English — choose their preferred voice, and hear the computer speak it out loud with realistic human-like pronunciation.

💡 Why This Project?

  • Real-World Application: TTS technology is used in Google Assistant, Alexa, Siri, navigation apps, and accessibility tools for visually impaired people.
  • AI & Cloud Computing: It demonstrates how cloud-based AI services can be accessed using Python.
  • Multi-Language Support: The program supports both Hindi and English, making it relevant for Indian users.
  • Interactive: It's not just a script — it takes user input and produces audible output.

✨ Key Features

  • 🔊 Converts any typed text into natural AI speech
  • 🇮🇳 Supports Hindi Male & Female voices
  • 🇺🇸 Supports English US Male & Female voices
  • 🇬🇧 Supports English UK Male & Female voices
  • 🎛️ User-friendly menu-driven interface
  • ✅ Input validation — handles wrong inputs gracefully
  • 🔁 Option to speak multiple texts without restarting
  • 🧹 Automatic cleanup of temporary audio files

1

Program Starts & Shows Menu

The program displays a welcome message and shows 6 voice options — Hindi Male/Female, English US Male/Female, English UK Male/Female. User picks one by entering a number.

2

User Enters Text

The user types any text they want to hear. It can be in Hindi (using Hindi keyboard) or English. The program validates that the text is not empty.

3

Text Sent to Microsoft's Cloud

The edge_tts library sends the text and selected voice to Microsoft's Azure cloud servers over the internet. The AI neural network processes the text.

4

AI Generates Speech

Microsoft's neural TTS engine converts the text into natural-sounding speech. It understands pronunciation, intonation, pauses, and emotions. The result is sent back as an MP3 file.

5

Audio is Played

The MP3 file is saved locally and played through the computer's speakers using the pygame.mixer module. The program waits until the audio finishes playing.

6

Repeat or Exit

The user can choose to speak another text or exit. On exit, the temporary MP3 file is automatically deleted to keep the system clean.

Source Code

Complete Python source code of the project

text_to_speech.py 
# ==========================================
#   TEXT-TO-SPEECH PROGRAM (Edge TTS)
#   CBSE Class 11 - Computer Science Project
#   Made by: Harsh | Class XI-A | Roll No: 17
# ==========================================

import edge_tts
import asyncio
import pygame
import time
import os

# ---------- AVAILABLE VOICES ----------
VOICES = {
    "1": ("Hindi Female",      "hi-IN-SwaraNeural"),
    "2": ("Hindi Male",        "hi-IN-MadhurNeural"),
    "3": ("English Female US", "en-US-JennyNeural"),
    "4": ("English Male US",   "en-US-GuyNeural"),
    "5": ("English Female UK", "en-GB-SoniaNeural"),
    "6": ("English Male UK",   "en-GB-RyanNeural"),
}

OUTPUT_FILE = "speech.mp3"


# ---------- GENERATE SPEECH ----------
async def generate_speech(text, voice):
    communicate = edge_tts.Communicate(text, voice)
    await communicate.save(OUTPUT_FILE)


# ---------- PLAY AUDIO ----------
def play_audio():
    pygame.mixer.init()
    pygame.mixer.music.load(OUTPUT_FILE)
    pygame.mixer.music.play()
    while pygame.mixer.music.get_busy():
        time.sleep(0.1)
    pygame.mixer.music.unload()
    pygame.mixer.quit()


# ---------- MAIN PROGRAM ----------
def main():
    print()
    print("  " + "=" * 40)
    print("      TEXT  TO  SPEECH  PROGRAM")
    print("      Powered by Microsoft Edge AI")
    print("  " + "=" * 40)

    # Show voice options
    print("\n  Available Voices:")
    print("  " + "-" * 35)
    for key, (name, _) in VOICES.items():
        print(f"    {key}. {name}")
    print("  " + "-" * 35)

    while True:

        # --- Choose Voice ---
        while True:
            choice = input("\n  Choose voice (1-6): ").strip()
            if choice in VOICES:
                break
            print("  Invalid! Enter 1 to 6.")

        voice_name, voice_id = VOICES[choice]

        # --- Enter Text ---
        text = input("  Enter text to speak: ").strip()
        if not text:
            print("  Text cannot be empty!")
            continue

        # --- Generate & Play ---
        print(f"\n  Generating speech in [{voice_name}]...")
        asyncio.run(generate_speech(text, voice_id))

        print("  Playing audio... 🔊")
        play_audio()
        print("  Done! ✅")

        # --- Play Again? ---
        again = input("\n  Speak again? (y/n): ").lower()
        if again != "y":
            break

    # Cleanup temp file
    if os.path.exists(OUTPUT_FILE):
        os.remove(OUTPUT_FILE)

    print("\n  Thanks for using! Goodbye! 👋\n")


# --- RUN ---
main()
